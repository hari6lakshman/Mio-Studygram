# **App Name**: Mio

## Core Features:

- AI-Powered Chat: Utilize Gemini 2.5 Flash via the @google/genai SDK to provide sophisticated, intelligent, and luxurious educational assistance, maintaining context through chat history.
- Markdown Parsing: Parse markdown-style bold text (**) within messages. Apply gold color (#d4af37) to Mio's bold text to highlight educational concepts, and white color to the user's bold text.
- Dynamic Input Area: Implement an auto-expanding textarea for input, providing a seamless and adaptive user experience.
- Gold Standard Clarification Tool: An AI-powered tool which enables Mio to determine when to highlight complex topics with gold text, enhancing educational understanding.

## Style Guidelines:

- Background: Deepest Midnight Blue (#050a14) for a luxurious feel.
- Primary Accent: Metallic Gold (#d4af37) to signify luxury and highlight key elements.
- Text: Cream (#f3e5ab) and Cornsilk (#fff8dc) for readability and elegance.
- Headings & Brand: 'Cinzel' (serif) for a classic, elegant look. Note: currently only Google Fonts are supported.
- Body Text: 'Lato' (sans-serif) for clean readability. Note: currently only Google Fonts are supported.
- Subtle backdrop blurs, glows, and custom scrollbar styling to add depth and visual interest.
- Thin gold borders (1px) and center-aligned header for a high-end feel.
